# CredAI — AI Spend Audit Tool

A premium, interactive AI Stack Audit & Optimization tool built for **Credex**. It helps startups and development teams inspect their AI tooling spend, find redundancies, identify downgrades, and discover alternative opportunities—driving high-value lead generation for Credex credits.

## Features
- **MVP 1:** Dynamic Multi-step AI Spend Input
- **MVP 2:** Deterministic Pricing & Audit Recommendation Engine (parsed directly from PRICING_DATA.md)
- **MVP 3:** Highly Shareable Audit Results Dashboard (screenshot-worthy)
- **MVP 4:** Personalized AI Summary powered by Anthropic's Claude API
- **MVP 5:** Value-First Lead Capture (gated after value delivery)
- **MVP 6:** Public Share Links with Custom Dynamic Open Graph Social Previews

## Setup & Running Locally
```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Run Vitest test suite
npm test
```
